Virtualenvi jaoks on kõik requirement-id failis requirements.txt. Pythoni versioon on 3.10.8.

1.Aktiveeri virtualenv (cd ilmarakendus_env/Scripts  ja siis .\activate)

2. pip install -r requirements.txt terminalis (kui pip install ei toimi kasuta python -m pip install

Kui nõuded täidetud siis saad runnida commandi python manage.py runserver
